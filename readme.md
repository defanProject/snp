# SnapTokDownload TikTok videos without watermark. Supports MP4 (HD), MP3 audio, and photo slideshows. No login, no watermark, no bullshit.Live: [snaptok.lol](https://snaptok.lol) | Updates: [t.me/snaptokUpdate](https://t.me/snaptokUpdate)## Features- Download TikTok video tanpa watermark (HD)- Download audio/musik MP3- Download foto slideshow- TikTok search by keyword- Trending page- Proxy download (stream langsung ke device)- Maintenance mode (via admin panel)- Server status monitoring (CPU, RAM, Disk)- Multi-language: EN, ID, RU, ZH, JA- Auto redirect snaptok.my.id ke snaptok.lol- Admin panel di `/admin/akses`## Stack- React 19 + TypeScript- Vite 6 + Tailwind CSS v4- Express backend- i18next (5 bahasa)- Firebase- TikWM API## Routes| Path | Keterangan ||------|-----------|| `/:lang` | Home / downloader || `/:lang/tiktok-search` | Search video || `/:lang/trending` | Trending || `/:lang/faq` | FAQ || `/:lang/terms` | Terms of Service || `/:lang/guide` | Guide || `/:lang/cookies` | Cookies || `/:lang/privacy` | Privacy Policy || `/:lang/status` | Server status || `/admin/akses` | Admin panel |## API Endpoints| Method | Path | Keterangan ||--------|------|-----------|| POST | `/api/tiktok/download` | Download video by URL || POST | `/api/tiktok/search` | Search video by keyword || GET | `/api/proxy` | Proxy stream/download file || GET | `/api/health` | Health check + system info || GET | `/api/admin/stats` | Stats admin || POST | `/api/admin/maintenance` | Toggle maintenance mode |## Run LocallyNode.js v20+ required.```bashgit clone https://github.com/youruser/snaptok.gitcd snaptoknpm installnpm run buildnpm run dev```Server jalan di `http://localhost:8080`.## Build & Deploy (VPS + PM2)```bashnpm run buildpm2 start tsx --name "snaptok" -- server.tspm2 savepm2 startup```Cloudflare Tunnel:```bashpm2 start cloudflared --name "snaptok-tunnel" -- tunnel runpm2 save```## Maintenance ModeAktifin/matiin via admin panel atau langsung edit `settings.json`:```json{ "maintenance": "yes" }```
